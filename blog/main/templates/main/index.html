Добро пожаловать!

Вы попали на главную страницу мини-блога, написанного начинающим разработчиком с помощью Python/Django.

В этом тренировочном проекте, реализовано:

  1. Связанные модели: статьи, категории, комментарии.
  2. Добавление, удаление, редактирование статей.
  3. Добавление комментариев к статье.
  4. Счетчик количества комментариев.
  5. Регистрация, авторизация.

Верстка сайта сделана по PSD шаблону с небольшими изменениями. Позиционирование с помощью FlexBox.

Для названий классов была выбрана BEM методология.
